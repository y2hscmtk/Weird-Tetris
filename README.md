가로3칸 세로3칸 대각선3칸으로 같은색의 블록이 놓여지면 해당 블록은 사라진다.
사라지는 블록 1개당 50점을 얻을수 있으며, 1000점, 2000점, 등 점수가 천단위를 넘길때마다 레벨과 블록이 떨어지는 속도가 증가한다.
